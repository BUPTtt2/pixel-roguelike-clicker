# Boss 全面重设计 - 设计文档

## 用户诉求
1. BOSS 必须看起来"可怕"（高血量 + 复杂技能）
2. 硬核玩家，必须走位躲技能
3. 难度要明显（武器不能过早结束战斗）
4. 多主题背景（绿/红/蓝/紫）
5. 复杂 BOSS：8+ 技能，快速连招
6. 修复 BOSS 卡在屏幕上问题
7. 修复地狱难度一团糟问题
8. 修复 UI 文字叠在一起问题

## 核心问题分析

### 问题 1: BOSS 卡在屏幕上（最严重）
**根因**：多血条系统的 `takeDamage` 方法中，当 `hp <= 0` 但还有下一条血时，调用 `this.spawnMinions()` 等但不重新启用 `update()`，加上 BOSS 死亡/重生时 tweens 没清理干净，导致：
- 旧 BOSS 的 container 残留
- 血条和名字残留
- 旧 tween 继续在新的 BOSS 上跑

**解决**：完全重构 BOSS 生命周期管理。

### 问题 2: BOSS 技能看不到
**根因**：
- 多数技能用 `particles.explosion()`，视觉一闪而过，玩家根本看不清
- 没有清晰的"预警 → 释放"两步动画
- 技能冷却太长（1500ms），BOSS 经常被秒杀前只放 1-2 个技能

**解决**：所有技能必须有：
1. **预警阶段（0.5-1.5秒）**：地面红色圆圈/激光线，玩家必须走位
2. **释放阶段**：明显的特效（粒子+屏幕震动+屏幕短暂变红）
3. **结算阶段**：伤害数字、爆炸

### 问题 3: 地狱难度一团糟
**根因**：
- 血量/伤害/速度倍率不够爆炸
- BOSS 没有真正的阶段切换（破段时只是数字变）
- 技能没有连击感

**解决**：地狱难度专属：
- BOSS 血量 ×3，伤害 ×2，速度 ×1.5
- 多血条数量从 5/8/12/15 → 8/15/20/25
- 真正的形态切换：BOSS 改变外观/颜色

### 问题 4: UI 文字叠在一起
**根因**：
- BOSS 名字 y=118，BOSS 精灵 size=100-230 大小不一
- 血条/分段指示器没考虑 BOSS 大小自适应
- 章节名/难度名/时间叠在一起

**解决**：
- BOSS 名字移到屏幕顶部固定位置（y=90），跟 BOSS 实际位置解耦
- 章节名和难度名合并显示
- 时间独占一行

### 问题 5: 背景没有
**解决**：每章一个主题色 + 简单的程序化背景元素（点状星空/烟雾/光斑）

## 设计方案

### 1. 全新 BOSS 生命系统

```
BOSS 状态机:
  IDLE → WARN → ACTIVE → BREAK → ACTIVE → ... → DEATH
  
  IDLE: 不在场景
  WARN: 玩家收到警告，准备登场
  ACTIVE: 正在战斗
  BREAK: 破段（短动画，不动）
  DEATH: 死亡动画
```

**关键改动**：
- `forceDestroy()` 改为 `destroyAllVisuals()` + `stopAllTweens()`
- BossManager 维护 `activeBosses` 数组（只允许 1 个 BOSS 在场）
- 每次 spawnBoss 前强制清理所有 BOSS 视觉

### 2. 全新技能系统

**技能模板**：
```javascript
bossSkill({
    name: '暗影斩',
    warning: { duration: 800, type: 'circle', radius: 200, color: 0x880044 },
    effect: { duration: 500, type: 'slash', damage: 1.2 },
    cooldown: 3000
})
```

**所有技能改造**：
- 每个技能都有 warning + effect 两段
- warning 阶段显示地面红圈/激光线
- effect 阶段：粒子+屏幕震动+伤害数字

**8 个 BOSS 全部重做技能**（每个 6-8 个技能）：
- 哥布林王：横扫、冲锋、扔金币、召唤哥布林、狂暴、毒云
- 骷髅领主：扔骨头、骨刺激光、骷髅军团、死亡绽放、骷髅盾、狂暴
- 暗影龙：俯冲、龙息、龙卷风、陨石雨、飞升、狂暴
- 深渊魔王：暗影斩、地狱火、召唤恶魔、诅咒领域、暗影步、狂暴
- 时空守护者：时间减速、时间停止、瞬移斩、暗影分身、时之结界、狂暴
- 元素领主：火浪、冰封、雷暴、地震、元素爆发、狂暴
- 虚空先知：幻象、心控、虚空白洞、克隆术、虚空回响、狂暴
- 暗影神：暗影风暴、死亡射线、虚空牢笼、召唤联军、湮灭、狂暴、升天

### 3. 难度系统重构

```javascript
const DIFFICULTY = {
    easy:   { hpMul: 1.0, dmgMul: 1.0, spdMul: 1.0, bars: 3,  skills: 'first-3', aiDelay: 2500 },
    normal: { hpMul: 1.8, dmgMul: 1.5, spdMul: 1.1, bars: 5,  skills: 'all',     aiDelay: 1800 },
    hard:   { hpMul: 3.0, dmgMul: 2.0, spdMul: 1.3, bars: 8,  skills: 'all',     aiDelay: 1300 },
    hell:   { hpMul: 5.0, dmgMul: 3.0, spdMul: 1.5, bars: 12, skills: 'all',     aiDelay: 800, combo: true }
};
```

**地狱难度专属**：
- BOSS 有 50% 概率连续放 2 个不同技能
- 暴击伤害 ×2
- 玩家受伤增加 20% 减速效果

### 4. 武器平衡（防止过早结束）

- 武器基础伤害降低 30%
- 暴击率从 15% 降到 8%
- 攻速减慢 20%
- 武器升级所需经验 +50%
- 这样地狱难度下 BOSS 会成为真正的挑战

### 5. UI 重构

```
顶部布局（绝对位置，固定不变）:
  y=15:  [时间 00:00]    [击杀 423]
  y=40:  [第 3 波]       [Boss: 0]
  y=60:  [第一章 暗影降临]    [难度: 地狱]
  
BOSS 专属:
  y=100: [BOSS名字 + 等级]  (屏幕顶部居中，不跟随 BOSS)
  y=125: [BOSS血条]
  y=150: [分段指示器 ■■◆□□ 2/5]
```

### 6. 背景系统

5 个主题对应 8 个章节：
- 第一章（哥布林王）：森林绿
- 第二章（骷髅领主）：灰墓
- 第三章（暗影龙）：黄昏紫
- 第四章（深渊魔王）：血红
- 第五章（时空守护者）：星蓝
- 第六章（元素领主）：熔岩橙
- 第七章（虚空先知）：虚空紫
- 第八章（暗影神）：纯黑+红

每个背景 = 基础色 + 50-100 个随机"星点"或"光斑"（每帧缓慢移动）。

## 实施步骤

1. **修复 BOSS 残留**（最优先）
   - 重构 BossManager
   - 强化 forceDestroy
   - 确保 spawnBoss 前彻底清理

2. **重做技能系统**
   - 实现 skill 模板
   - 重做 8 个 BOSS 的所有技能（48+ 个技能）
   - 每个技能都有 warning + effect

3. **重构难度系统**
   - 新 DIFFICULTY 表
   - 地狱难度专属行为

4. **UI 重构**
   - 固定顶部布局
   - 文字分层清晰

5. **背景系统**
   - 8 主题背景

6. **武器平衡**
   - 降基础伤害
   - 升升级难度

7. **测试与推送**

## 不做什么
- 不加新武器
- 不加新宠物
- 不改商店系统
- 不加新章节/剧情
- 不动 BOSS 数量（保持 8 个）

## 风险
- 大量代码改动，可能引入新 bug
- 文件已 400KB，重构要小心
- 每改一步都要测
