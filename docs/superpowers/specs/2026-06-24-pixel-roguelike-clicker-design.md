# 像素点击肉鸽游戏设计文档

## 一、游戏概述

### 1.1 游戏名称
**像素地牢猎手** (Pixel Dungeon Hunter)

### 1.2 游戏类型
点击挂机肉鸽游戏 / Roguelike Clicker

### 1.3 核心玩法
玩家点击屏幕攻击敌人，敌人死亡掉落金币和经验。敌人存在超时会对玩家造成伤害。击败Boss获得奖励选择，通过构筑能力存活更长时间。

---

## 二、游戏架构

### 2.1 技术栈
- **引擎**: Phaser 3 (HTML5 Canvas)
- **语言**: JavaScript ES6+
- **样式**: CSS3 (像素风格)
- **存储**: LocalStorage (元进度)

### 2.2 场景结构
```
BootScene       → 资源加载
MenuScene       → 主菜单/角色选择
GameScene       → 主游戏场景
RewardScene     → Boss奖励选择
UpgradeScene    → Boss战武器升级
GameOverScene   → 结算界面
```

### 2.3 核心系统
| 系统 | 职责 |
|------|------|
| EntityManager | 管理玩家、敌人、随从等实体 |
| CombatSystem | 处理攻击、伤害、碰撞 |
| RewardSystem | Boss奖励生成与选择 |
| ProgressionSystem | 经验、等级、专精点 |
| EconomySystem | 金币管理、武器升级 |
| WaveSystem | 敌人波次与Boss生成 |

## 三、角色与武器系统

### 3.1 初始武器选择（5款）

| 武器名称 | 攻击方式 | 伤害 | 攻速 | 特色 |
|----------|----------|------|------|------|
| 短剑 | 单体点击 | 10 | 快 | 基础武器 |
| 战斧 | 范围AOE | 25 | 慢 | 伤害高，范围大 |
| 法杖 | 穿透射击 | 15 | 中 | 直线穿透多个敌人 |

### 3.2 武器升级系统
- **升级时机**: 仅在击败Boss后
- **升级消耗**: 金币
- **升级效果**: 伤害提升20%，最高5级

## 四、随从系统

### 4.1 随从列表（2款，隐藏解锁）

| 随从名称 | 解锁条件 | 效果 |
|----------|----------|------|
| 光明精灵 | 击败第3个Boss | 每5秒回复玩家5%HP |
| 骷髅战士 | 累计存活时间超过10分钟 | 自动攻击附近敌人，伤害=玩家攻击30% |

### 4.2 随从获取
- 在Boss奖励中以**稀有奖励**形式出现
- 解锁后，奖励池中加入该随从

## 七、奖励系统

### 7.1 Boss奖励池

**普通奖励（70%权重）**:
- 金币 +50~100
- 经验 +50~100
- 攻击专精 +1
- 生命专精 +1
- 金币专精 +1

**稀有奖励（25%权重）**:
- 暴击专精 +1
- 护甲专精 +1
- 经验专精 +1
- 随从专精 +1
- 武器进化材料

**史诗奖励（5%权重）**:
- 已解锁随从
- 武器直接+1级
- 所有专精+1
- 最大HP +20

### 7.2 奖励选择界面
- 宝箱开启动画
- 3个随机奖励展示
- 点击选择

## 九、数值平衡

### 9.1 难度曲线
- 游戏时间越久，敌人HP和数量递增
- 每60秒敌人属性提升10%
- Boss属性随波次提升20%

### 9.2 成长曲线
- 基础攻击成长: 专精+武器升级
- 基础生存成长: HP上限+护甲+回血随从
- 经济成长: 金币掉落加成

---

## 十、特色亮点

1. **像素复古美术**: 经典8-bit风格，快速生成素材
2. **策略构筑**: 通过专精点和奖励选择打造独特Build
3. **隐藏随从**: 增加探索和重玩价值
4. **渐进难度**: 难度随时间平滑递增
5. **一键升级**: Boss后武器升级，满足成长快感

## 十一、AI协作开发

### 11.1 AI应用场景
| 模块 | AI作用 |
|------|--------|
| 代码生成 | 游戏核心逻辑、系统架构 |
| 素材生成 | 角色精灵、敌人、道具图标 |
| 音效生成 | 点击音效、战斗音效 |
| 数值平衡 | 自动计算和调整数值参数 |
| 测试验证 | 自动测试游戏逻辑 |

### 11.2 开发流程
1. AI生成游戏框架和核心系统
2. AI生成像素美术素材
3. AI生成音效和音乐
4. AI辅助数值调优
5. 人工测试和微调
