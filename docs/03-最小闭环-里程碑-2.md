# 里程碑2: 游戏体验增强

> 完成时间: 2026-06-24

## 实现内容

### 1. 粒子效果系统 (ParticleManager)

| 效果 | 描述 |
|------|------|
| 金币粒子 | 💰飞向金币显示区域 |
| 经验粒子 | ✨飞向经验条 |
| 击杀粒子 | 8方向彩色爆炸 |
| Boss击败粒子 | 20方向大爆炸+中心闪光 |
| 升级粒子 | 12颗⭐向外扩散 |
| 回血粒子 | ❤️向上飘动 |
| 随从攻击粒子 | 灰色圆形扩散 |

### 2. 敌人移动动画

| 敌人类型 | 移动速度 (像素/秒) |
|----------|-------------------|
| 史莱姆 | 30 |
| 蝙蝠 | 60 |
| 骷髅 | 40 |
| 幽灵 | 50 |
| 石像鬼 | 25 |

敌人向玩家位置移动，距离50像素内停止。

### 3. 暂停功能和音量控制

| 功能 | 实现 |
|------|------|
| 暂停按钮 | ⏸️/▶️切换，显示暂停覆盖层 |
| 音量按钮 | 🔊/🔇切换，开关音效 |
| 暂停覆盖层 | 半透明黑色背景+提示文字 |

### 4. 游戏数值平衡调整

| 参数 | 原值 | 新值 | 说明 |
|------|------|------|------|
| 初始HP | 100 | 150 | 增加容错空间 |
| 初始经验阈值 | 50 | 30 | 更快升级体验 |
| Boss出现时间 | 60s,120s,180s... | 45s,90s,150s... | 更早遇到Boss |
| 史莱姆HP | 30 | 25 | 更易击杀 |
| 史莱姆超时 | 8s | 6s | 更快造成伤害 |
| 蝙蝠HP | 20 | 15 | 更易击杀 |
| 蝙蝠超时 | 6s | 5s | 更快造成伤害 |
