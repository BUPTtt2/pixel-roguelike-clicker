## 实现内容

### 1. 性能优化

| 功能 | 描述 |
|------|------|
| 敌人对象池 | EnemyManager添加pool数组，复用敌人对象 |
| getEnemyFromPool | 从池中获取空闲敌人 |
| returnToPool | 敌人死亡后返回池而非销毁 |
| Enemy.reset | 重置敌人状态用于复用 |
| 像素背景优化 | 简化背景渲染，减少对象数量 |

### 2. 像素风格背景和界面美化

| 功能 | 描述 |
|------|------|
| index.html美化 | 像素风格边框和阴影 |
| 背景星星效果 | 15个闪烁星星，营造氛围 |
| 游戏边框 | 深色边框装饰 |
| 背景色调整 | 统一暗色主题 |

### 3. 更多音效

| 音效 | 描述 |
|------|------|
| 武器专属音效 | 短剑/战斧/法杖/弓箭/魔杖各有独特音效 |
| 金币拾取音效 | 击杀敌人获取金币时播放 |
| 随从攻击音效 | 骷髅战士攻击时播放 |
| 随从回血音效 | 光明精灵回血时播放 |
