## 本地运行

下载项目后，进入目录执行：

```bash
python -m http.server 8080
# 浏览器打开 http://localhost:8080/index.html
```

或使用 Node.js：
```bash
npx http-server -p 8080
```

> 无需安装任何依赖，打开即玩。推荐 Chrome / Edge 浏览器。

## 核心玩法

### 武器系统

5 种武器，各有独特攻击模式：

| 武器 | 攻击类型 | 核心机制 |
|------|---------|---------|
| 暗影剑 | 近战扇形 | 流血 + 吸血 |
| 烈焰斧 | 近战大范围 | 燃烧 + 击晕 + AOE |
| 冰霜法杖 | 远程直线 | 减速 + 冰锥分裂 |
| 狩猎弓 | 远程弹道 | 护甲穿透 + 暴击双发 |
| 奥术魔杖 | 追踪弹射 | 元素共鸣 |

### 敌人与 BOSS 系统

**8 个章节 BOSS**，每个 6+ 招技能：

| 章节 | BOSS | 核心技能 |
|------|------|---------|
| 1 | 哥布林王 | 横扫 + 召唤 + 金币雨 |
| 2 | 骷髅领主 | 骨弹 + 骷髅军团 + 死亡绽放 |
| 3 | 暗影龙 | 飞行 + 龙息 + 龙卷风 + 陨石 |
| 4 | 深渊魔王 | 暗影斩 + 地狱火 + 召唤恶魔 |
| 5 | 时空守护者 | 时间减速 + 时间停止 + 瞬移打击 |
| 6 | 元素领主 | 火焰波 + 冰河世纪 + 雷暴 + 地震 |
| 7 | 虚空先知 | 幻象 + 心控 + 虚空白洞 + 克隆 |
| 8 | 暗影神 | 暗影风暴 + 死亡射线 + 虚空牢笼 + 湮灭终极 |

**4 档难度**：
- 简单：单技能 + 2 招式
- 普通：完整技能池
- 困难：25% 概率组合技
- 地狱：40% 概率三连击 + 隐藏阶段

### 宠物系统

5 大宠物，环绕玩家飞行并自动攻击：

- 光明宠物：基础攻击
- 暗影宠物：高暴击
- 火焰宠物：燃烧效果
- 寒冰宠物：减速控制
- 闪电宠物：连锁闪电

### 探索系统

- **5 个区域**：腐朽森林 → 沉没沼泽 → 焦土荒原 → 永夜城堡 → 裂隙深处
- **多主题背景**：每章节切换主题色（绿/灰/紫/红/蓝/橙/深紫/黑红）+ 浮动光斑
- **符文阵**：靠近激活 + 击败守卫 + 获得护盾/狂暴/稀有掉落
- **迷你地图**：实时显示敌人/宝箱/BOSS 位置
- **宝箱**：固定位置，靠近 65px 拾取

## 技术架构

| 项 | 说明 |
|---|---|
| 引擎 | Phaser 3.60（本地化，无构建步骤） |
| 架构 | 单文件 game_v5.js（约 9000 行） |
| 视觉资源 | 纯代码 Canvas 程序化生成（无外部图片） |
| 存档 | localStorage 自动保存 |
| 部署 | GitHub Pages 自动部署 |

### 核心机制

- **技能预警系统**：所有 BOSS 技能都有 0.5-1.5 秒预警（地面红圈/激光线），玩家必须走位躲避
- **多血条系统**：BOSS 有 3-12 段血条，分段指示器 ■■◆□□ 2/5
- **BOSS 阶段切换**：破段动画 + 难度提升 + 视觉特效
- **纯代码生成像素**：所有角色/BOSS/武器/宠物用 Graphics 动态绘制

## AI 协作开发

本项目从设计到实现全程使用 AI 协作完成。

### 模块化 AI 开发

| 模块 | AI 开发方式 |
|------|------------|
| 架构设计 | AI 设计 Boss/Enemy/Pet/Projectile/Particle 独立分层架构 |
| 像素美术 | AI 用 fillRect/fillCircle 逐像素绘制 200+ 精灵 |
| 技能系统 | AI 设计 skillWithWarning() 通用模板（预警+效果+屏幕震动），复用 48+ 个技能 |
| BOSS 行为树 | AI 设计状态机 WARN→ACTIVE→BREAK→DEATH |
| 物理碰撞 | AI 实现基于距离判断的 AABB 碰撞，替代 Phaser 内置 Arcade Physics |
| 难度平衡 | AI 调参血量倍率、伤害倍率、技能冷却、技能组合概率 |
| 调试修复 | AI 通过静态分析+浏览器回放定位 20+ 个 Bug |
| Git 部署 | AI 自动 commit+push 到 GitHub Pages，加 .nojekyll 绕过 Jekyll |

### AI 核心价值

1. **极快迭代**：每个功能 30 秒内完成
2. **跨模块一致性**：维护全局设计语言（美术风格/UI 字号/配色/命名）
3. **Bug 模式识别**：通过现象反推根因（如 BOSS 残留 = 无限 tween 阻止 container 销毁）
4. **数据驱动平衡**：基于游戏内事件（连击数/击杀数/死亡时间）调参
5. **文档同步**：改代码同时更新设计文档
