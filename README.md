## 存档管理

游戏支持完整的存档管理系统（主菜单 → 存档管理）：

- **查看详情**：12 项存档数据 + 存档大小
- **导出存档**：复制 JSON 到剪贴板，便于备份
- **导入存档**：粘贴 JSON，自动校验格式
- **删除存档**：二次确认，防误删
- **加载体验存档**：一键解锁全内容，供快速体验

> 存档存储于浏览器 localStorage，不会上传服务器。

- **BOSS 纹理**：每个 BOSS 通过几何图形组合生成独特外观（皇冠、骨十字、龙翼等）

- **特效系统**：粒子效果、屏幕震动、伤害飘字、暴击闪光

## AI 协作开发

本项目从设计到实现全程使用 AI 协作完成。

### 模块化 AI 开发

| 模块 | AI 开发方式 |
|------|------------|
| SaveData 存档系统 | 描述数据结构需求，AI 生成 load/save/clear/default + 字段定义 |
| EnemyManager | 描述"渐进式波次、精英分级、无空窗"，AI 生成刷怪逻辑 |
| BossManager | 描述"8 BOSS、多段血条、防无限分裂"，AI 生成配置和技能链 |
| WeaponFactory | 描述"5 武器、4 阶段、形态进化"，AI 生成 configs 和伤害计算 |
| 程序化纹理 | 描述"独特外观"，AI 生成 generateBossTextures() 函数 |

### AI 核心价值

1. **代码生成**：一次性产出 3000+ 行可运行主逻辑
2. **根因定位**：从"看不见地图"定位到"红色叠加层 alpha 过高"
3. **方案取舍**：text_to_image API 失败时主动切换为程序化生成
4. **数值平衡**：建立 TTK 基准，多轮调优攻速/掉落/吸血/专精
5. **迭代速度**：反馈到部署 < 10 分钟/轮
