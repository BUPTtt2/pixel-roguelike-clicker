# 武器与经济系统全面重构方案

## Context

用户反馈武器都太弱、没有晋升机制；金币买经验卷轴不合理；金币给太多；元素特性看不出来；小怪掉落不合理；升级应该越来越难。经代码审查还发现4个关键bug（武器经验全局共享、弓/魔杖击杀不给经验、双武器等级系统混用、弓护甲穿透未实现）。需要全面重构武器、经济、经验、掉落四大系统。

所有改动集中在单文件 `d:\Appt\大三下\学习\vue,fastAPI,langchain\game_v5.js`。

---

## 第一步：修复4个关键Bug

### Bug 1：武器经验全局共享（L3786-3800）
`weaponExp` 是标量，切换武器会带走进度。
- 改 `weaponExp` 为对象 `{sword:0, axe:0, staff:0, bow:0, wand:0}`
- `addWeaponExp` 中改为 `this.weaponExp[weaponType]`
- 初始化处（L2339, L2919, L2981）同步改为对象

### Bug 2：弓/魔杖击杀不给+2武器经验
- `performLineAttack`（L3727-3731）：`hits.forEach` 内加 `if (died) this.addWeaponExp(type, 2)`
- `Projectile.hitEnemy`（L678-689）：击杀时调用 `this.scene.addWeaponExp(type, 2)`

### Bug 3：双武器等级系统混用
- 删除 `runtime.weaponLevel`，所有引用改为 `this.weaponLevels[this.runtime.weaponType]`
- 影响行：L3568(getDamage调用)、L3813(getSkillDamage调用)、L4067(onLevelUp)、L4279(claimBossReward weaponUp)、L4318(newWeapon fallback)

### Bug 4：弓护甲穿透未实现
- `Enemy.takeDamage`（L999）增加 `armorPierce` 参数
- `applyElementEffect`（L3687）和 `Projectile.hitEnemy`（L678）传入 `armorPierce`

## 第二步：武器晋升系统

### 武器经验曲线（L3791）
```
need = cur <= 3 ? Math.floor(40 * 1.4^(cur-1))      // 40, 56, 78
      : cur <= 6 ? Math.floor(120 * 1.5^(cur-4))    // 120, 180, 270
      : Math.floor(350 * 1.6^(cur-7))               // 350, 560, 896, 1434
```

### 5种武器10级成长路线（3阶段：基础Lv1-3 / 进阶Lv4-6 / 终极Lv7-10）

**暗影剑**：流血+吸血+连击
- Lv4质变：流血可叠加3层
- Lv7质变：连击系统——每次命中减CD 15ms，满10层下次必暴击
- Lv10满级：击杀流血敌人触发范围血爆

**烈焰斧**：击晕+燃烧+AOE
- Lv4质变：攻击产生火焰残留区域(2秒)
- Lv7质变：蓄力机制——按住1秒释放3倍伤害+击退
- Lv10满级：击晕敌人被引爆，AOE燃烧伤害

**冰霜法杖**：减速+控场+多弹
- Lv4质变：冰锥命中后分裂为2个小冰锥
- Lv7质变：减速50%+的敌人20%概率冻结1.5秒
- Lv10满级：冻结结束时额外碎冰伤害

**狩猎弓**：穿透+暴击+护甲穿透
- Lv4质变：暴击不消耗穿透次数
- Lv7质变：标记系统——命中敌人被标记，对其+30%伤害
- Lv10满级：穿透流——每穿一个敌人伤害+15%（最多+75%）

**奥术魔杖**：弹射+追踪+多目标
- Lv4质变：弹射留下奥术残留，1秒后爆炸
- Lv7质变：奥术共鸣——同时命中3+敌人触发链式闪电
- Lv10满级：每第5次攻击自动释放免费技能

### 代码修改要点
- `Weapon` 类（L108-203）新增方法：`getBleedStacks`、`getComboSystem`、`getFireField`、`getSplit`、`getFreezeChance`、`getMark`、`getResonance` 等
- `getElementEffect`（L3597-3622）增加质变标志字段
- 各 `perform*Attack` 方法根据质变标志实现新机制
- `Projectile` 类（L573-697）增加标记/穿透递增/残留等逻辑
